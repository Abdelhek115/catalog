Twitter's terms of service don't allow tweet datasets to be distributed, but they do allow tweet id datasets to be shared. This honors content creators right to be forgotten, while allowing researchers to share their research data.

This site is a catalog of datasets that are publicly available. If you would like to turn these datasets back into JSON use the Hydrator desktop application, or Twarc if you are comfortable working at the command line.

If you would like to add your own dataset to the catalog please send us a pull request on GitHub.
