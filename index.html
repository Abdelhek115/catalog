Twitter's terms of service don't allow tweet datasets to be published on the web, but they do allow tweet identifier datasets to be shared. This speaks to content creators' right to be forgotten, while also allowing researchers to share their data with others.

This site is a catalog of datasets that are publicly available. If you would like to turn these tweet identifier datasets back into JSON you can use the Hydrator desktop application, or Twarc if you are comfortable working at the command line.

If you would like to add your own dataset to the catalog please follow these instructions.
